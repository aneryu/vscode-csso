# csso README

This extension is to format your code by [csso](https://github.com/css/csso).

## Features

It format this
```
.foo { color: #ff0000; }
.bar { color: rgba(255, 0, 0, 1); }

.foo { color: #ff0000; }
.bar { color: rgba(255, 0, 0, 1); }
```
to this
```
.bar,
.foo {
    color: red
}
```
